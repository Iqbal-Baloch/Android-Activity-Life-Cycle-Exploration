# Explore Android Activity Life cycle

simple android app to explore android activity life cycle

### Activity 1
Create an app that explores the life-cycle of an activity.

Write definition of all these methods and observe the results.

onCreate()
onStart()
onResume()
onPause()
onStop()
onRestart()
onDestroy()

### Observations
1. What callbacks are called when an app is first lunched?

2. what callbacks are called when home is pressed?

3. what callbacks are called when an app is restared from the luncher?

4. what callbacks are called when the is rotated?
